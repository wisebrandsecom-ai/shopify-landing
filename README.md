# EPB Skills

Skills para Claude Code desarrolladas por Wise Brands Ecom.

## Instalación

```bash
git clone https://github.com/wisebrandsecom-ai/EPB-Skills ~/.claude/skills/epb-skills
```

## Actualización

```bash
cd ~/.claude/skills/epb-skills && git pull
```

## Skills disponibles

| Skill | Descripción |
|-------|-------------|
| `shopify-landing` | Crea landings completas en Shopify replicando la estructura de un competidor |
| `competitor-ad-recreator` | Recrea ads estáticos de competidores con tu branding usando Higgsfield |
| `native-ad-recreator` | Recrea ads nativas con storytelling usando Higgsfield Soul |
| `competidores` | Scraper de Meta Ad Library — exporta competidores a Excel con sus mejores creativos |

## Uso

Cada skill se activa en Claude Code con:

```
Read the skill from: ~/.claude/skills/epb-skills/NOMBRE-SKILL/SKILL.md
```
